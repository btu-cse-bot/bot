# cse-discord-bot

## Installation 

- Create an `.env` file and add a line:
  
   ``` DC_TOKEN = [TOKEN_ID]```
- Install packages: 
  
  ```
  cd bot
  python3 -m pip install -r requirements.txt
  ```
  or
  ```
  cd bot
  pip3 install -r requirements.txt
  ```
## Run the bot:
 This code will run the bot and create all database tables.:
  ```
  python3 main.py
  ```
